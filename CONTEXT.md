# Odoo 19
This folder contains custom addons for Odoo 19.
For documentation on how to create custom addons for Odoo 19, see: https://www.odoo.com/documentation/19.0/developer.html

# Sports Federation
All addons starting with sports_federation are addons to manage our sports federation. They will help us to manage our clubs, tournaments, referees, rules...
