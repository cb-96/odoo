# Sports Federation Import Tools

## Purpose
Wizard-driven CSV import tools for clubs, teams, players, and tournament participants

## Dependencies
- `sports_federation_base`
- `sports_federation_people`
- `sports_federation_tournament`

## Models
This module contains only wizard models for import operations.
